# Cinema Tonight 🍿

A playful single-page invitation asking your friends: **Are we going to the cinema tonight?**

Built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- Dark cinema theme with animated gradients and floating emojis
- **YES** button with confetti celebration
- **NO** button that dodges, shakes, jumps, and teleports away
- Fully responsive for mobile (perfect for WhatsApp sharing)
- Zero backend — static site ready for Vercel

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Push to GitHub

```bash
git init
git add .
git commit -m "Add Cinema Tonight invitation page"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/cinema-tonight.git
git push -u origin main
```

## Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub.
2. Click **Add New Project** and import your `cinema-tonight` repository.
3. Vercel auto-detects Vite — no extra configuration needed.
4. Click **Deploy**.

Your live URL will be ready in seconds. Share it on WhatsApp!
